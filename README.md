# 🍽️ Restaurant Billing System (C++)

A simple console-based Restaurant Billing System developed in C++ to practice programming fundamentals and billing system logic.

## 🚀 Features

- Menu-driven interface
- Multiple item ordering system
- Input validation (item & quantity)
- Automatic bill calculation
- Item-wise quantity tracking
- GST calculation
- Discount system
- Formatted receipt generation using iomanip

## 🛠️ Technologies Used

- C++
- iostream
- iomanip

## 📚 Concepts Used

- Loops (do-while)
- Conditional Statements (if-else)
- switch-case
- Variables and calculations
- Output formatting
